## Install
```javascript
$ npm install --save @cloudelements/cardigan
```

## Stringified Function Support
* `function (a) ...`
* `async function (a) ...`
* `(a, b) => ...`
* `async (a, b) => ...`
* `a => ...`
* `async a => ...`
